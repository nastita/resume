# Company Logos

To add company logos to your resume, download the following images and save them in this directory with the specified names:

## Logo Files Needed

1. **lukso.png** - Universal Everything / LUKSO
   - Source: https://s2.coinmarketcap.com/static/img/coins/200x200/5625.png

2. **southworks.png** - SOUTHWORKS
   - Source: https://openqube.io/wp-content/uploads/2016/12/southworks-squarelogo-1567711527273-200x200.png

3. **halborn.png** - Halborn
   - Source: Search for "Halborn logo" or use company website

4. **poap.png** - Proof of Attendance Protocol
   - Source: https://d392zik6ho62y0.cloudfront.net/images/poap-logo.png

5. **kincarta.png** - KIN + CARTA
   - Source: https://media.designrush.com/agencies/345890/conversions/logo-logo-profile.jpg

## Instructions

1. Download each logo from the sources above
2. Save them with the exact filenames listed (e.g., `lukso.png`, `southworks.png`)
3. Place them in this `images/` directory
4. The resume will automatically include the logos when compiled

## Note

The LaTeX template is designed to work with or without logos. If a logo file is not found, it will simply display the company name without the logo.
